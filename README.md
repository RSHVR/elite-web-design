# Elite Web Design

A collection of specialized skills for Claude Code that enable premium, award-winning frontend development.

## Skills

| Skill | Description | Lines |
|-------|-------------|-------|
| **elite-design-core** | Design philosophy, visual hierarchy, spacing, typography, color | ~2,000 |
| **elite-gsap** | Complete GSAP reference (all plugins free since 2026) | ~3,500 |
| **elite-css-animations** | Scroll-driven API, View Transitions, @property | ~2,000 |
| **elite-layouts** | Bento grids, horizontal scroll, sticky/parallax, container queries | ~3,400 |
| **elite-performance** | 60fps optimization, Vite setup, debugging | ~2,500 |
| **elite-accessibility** | prefers-reduced-motion, WCAG, focus management | ~1,600 |
| **elite-inspiration** | Curated Awwwards/FWA references by pattern | ~2,500 |

**Total: ~17,700 lines** of specialized knowledge

## Installation

```bash
npx skills add RSHVR/elite-web-design
```

Or clone manually:

```bash
git clone https://github.com/RSHVR/elite-web-design.git ~/.claude/skills/elite-web-design
```

## Key Features

- **Granular loading** — Only relevant skills load based on your query
- **Accessibility-first** — `prefers-reduced-motion` baked into every pattern
- **Framework-agnostic** — Vanilla JS with React/Vue/Svelte integration notes
- **Decision frameworks** — CSS vs GSAP, layout selection by project type
- **Real references** — Actual Awwwards/FWA sites to study

## Triggers

Each skill activates on specific keywords:

- `"GSAP"`, `"ScrollTrigger"`, `"SplitText"` → elite-gsap
- `"bento grid"`, `"horizontal scroll"` → elite-layouts
- `"scroll-driven"`, `"view transitions"` → elite-css-animations
- `"60fps"`, `"performance"`, `"Vite"` → elite-performance
- `"reduced motion"`, `"accessibility"` → elite-accessibility
- `"inspiration"`, `"Awwwards"` → elite-inspiration
